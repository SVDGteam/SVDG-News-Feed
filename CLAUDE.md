## Project context
This is one of three internal hub repos for Silicon Valley Defense Group (SVDG),
deployed on Vercel. The other two repos are SVDGteam/svdg-hub and
SVDGteam/SVDG-News-Feed (or svdg-merry-go-round — adjust per repo).

## Brand system
Colors: dark navy #020633, green #22ff9b
Fonts: Space Grotesk, Lato, IBM Plex Mono

## Rules
- Never hardcode secrets, API keys, or credentials — use environment variables only
- Do not modify .env files
- Do not push or commit changes — stage only, let me review first
- Preserve existing component naming conventions
- When in doubt about a change, flag it as a recommendation instead of making it
